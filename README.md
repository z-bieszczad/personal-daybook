Personal Daybook is a simple web application for managing daily ideas, tasks and a shopping list.
The project was created as part of learning Spring Boot and full-stack application fundamentals.


## Technology Stack

- Java 17  
- Spring Boot  
- Spring MVC  
- Thymeleaf  
- H2 Database  
- HTML5 / CSS3  
- Bootstrap  

## Project Purpose

This project is intended to practice:

- CRUD operations  
- MVC architecture  
- Spring Boot fundamentals  
- Frontend integration with Thymeleaf
